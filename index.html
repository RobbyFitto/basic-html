Pendahuluan

Kali ini saya akan membuat documentation tentang html, yang dimana saya tau diri, saya masih banyak salah dalam penulisan html, dan ini menjadi pengingat dan basic. Sehingga tidak ada penggunaan CSS maupun JavaScript, mungkin ada di documentation yang terpisah. Terimakasih telah membaca ^-^

HTML atau Hypertext Markup Language merupakan bahasa markup yang dimana didalamnya itu terdapat serangkaian markup atau tag-tag, dengan fungsi untuk kasih tau browser, bagaimana sebuah kontent akan ditampilkan seperti text, paragraf ataupun yang lain.

HTML pertama kali diciptakan oleh TIm Berners Lee, selain membuat html, beliau juga menemukan banyak hal seperti WWW, HTTP, Browser dan lain-lain. Beliau menjadi direktur salah satu organisasi yang bernama World Wide Web Consortium / W3C. Kegiatan utama W3C sangat mulia, yaitu dimana membuat peraturan dan standar perkembangan web dalam jangka panjang. W3C tidak ada HQ tetapi terdiri dari beberapa gabungan institusi seperti MIT, France, Jepang dan Cina.

Jika ada tag baru, tag lama yang hilang itu berasal dari W3C berdasarkan peraturan yang baru dan diterapkan di web browser, sehingga web browser perlu diupdate untuk menikmati perkembangan yang ada. Bisa dilihat di Evolution of The Web (saat ini gabisa diakses), yang dimana menjelaskan bahwa dimulai html ver 1, browser pertama, teknlogi dan perkembangan hingga sampai saat ini.

Kali ini akan difokukan ke Web Design and Application yang didalamnya ada HTML CSS dan JavaScript, dengan memiliki fungsi-fungsi tersendiri. Contohnya sebegai berikut :

  • HTML dianalogikan sebagai sebuah kerangka atau manekin
  • Agar kerangka lebih menarik, dapat diberi baju, celana, warna ukuran dan lain-lain, yang dibantu oleh CSS
  • Manekin yang sudah menarik, agar lebih interaktif seperti berjalan, menyapa dan sebagainya dengan bantuan JavaScript

Di bagian Hello World yang terdapat gambar dasar code html, ada tag head yang didalamnya ada beberapa tag lain seperti:

Table

Table pada html digunakan untuk menampilkan data, sama halnya membuat tabel di ms excel dan word sesuai dengan kerjaan. ada satu hal lain yakni, jangan menggunakan tabel untuk membuat layout halaman (nanti dipelajari di CSS)

Pada tabel terdapat :

  • Box yang horizontal merupakan baris
  • Box yang vertikal merupakan kolom
  • Perpotongan baris dan kolom merupakan cell / data
  • Tiap baris dan kolom terdapat header

Tabel dibuat menggunakan tag table dan mempunyai tag-tag pendukung lain seperti berikut:

  • tag tr atau table row untuk membuat baris
  • tag td atau table data untuk membuat kolom
  • tag th atau table header buat header baik secara baris dan kolom
  • tag thead atau table header
  • tag tbody atau table body

Syntax table nya seperti ini:

Contoh table sederhana seperti berikut:

Baris 1, Kolom 1 Baris 1, Kolom 2
baris 2, kolom 1 baris 2, kolom 2

Syntax table nya seperti ini:

Contoh table komplex seperti berikut

Kolom 1 Kolom 2
Baris 1 Baris 1, Kolom 1 Baris 1, Kolom 2
baris 2 baris 2, kolom 1 baris 2, kolom 2

Jika ingin menambahkan border, space antar data dan lain-lain, bisa menggunakan atribut lain seperti:

  • border untuk memberi border di table
  • cellspacing menambah jarak antar cell / data
  • cellpadding menambah luas pada sebuah cell

Contoh penggunaannya sebagai berikut:

Baris 1, Kolom 1 Baris 1, Kolom 2
baris 2, kolom 1 baris 2, kolom 2

Selain itu ada 2 atribut lain seperti colspan dan rowspan. Dibawah ini contoh penggunaannya

rowspan 3 colspan 2
2, 2 2, 3
3, 2 3, 3
Hyperlink

Link adalah sebuah koneksi dari sebuah sumber ke sumber lain. Link pada html dibuat dengan tag a.

Tag a atau anchor/jangkar. Di analogikan seperti jangkar yang terhubung dengan dengan kapal, begitu juga pula dengan web pasti terhubung dengan sumber lain. Hyperlinkd dibagi menjadi 2 yaitu :

  • External link, yang dimana menghubungkan dengan link luar. Misalnya seperti, saya sedang belajar coding baik HTML CSS dan JavaScript di freeCodeCamp dan di Web Programming UNPAS di Youtube
  • Internal link, yang dimana menghubungkan dengan link di file yang sama. Contohnya seperti penggunaan navbar dibagian paling atas
Form

Di html juga bisa membuat form. Form itu sebuah elemen pada html yang digunakan untuk mengelola data yang dari input oleh user.

Form sendiri digunakan berbagai macam kebutuhan seperti, form registrasi, form login dan lain-lain, bahkan terdapat juga ada radio button, dropdown maupun checkbox. Form sendiri dibuat dengan tag form adapun tag atau elemen yang membantu pembuatan form yaitu:

  • input, bisa membuat textfield radiobutton ataupun dropdown
  • textarea, mirip seperti textfield tetapi lebih besar biasanya untuk beri pesan
  • select, membuat dropdown atau combobox
  • button, tombol untuk mengirimkan data atau yang lain
  • label, untuk menandai sebuah elemen input

Input sendiri memiliki atribut yang banyak seperti text, password, radio, checkbox dll
